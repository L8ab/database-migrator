A powerful database migration tool with multi-database support, version control, and rollback capabilities.

## Features

- Multi-database support (PostgreSQL, MySQL, MongoDB, SQLite)
- Version control for migrations
- Rollback capabilities
- Schema diffing
- Migration testing
- Transaction support
- CLI and programmatic API

## Installation

```bash
npm install -g @l8ab/database-migrator
```

## Usage

```bash
# Create a new migration
migrator create <migration-name>

# Run migrations
migrator up

# Rollback migrations
migrator down

# Check migration status
migrator status
```
